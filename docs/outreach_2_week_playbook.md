# 2-Week Outreach Playbook (Internal)

## Objective
Create a repeatable 14-day pipeline that produces qualified calls for Setta's paid creative modeling offer.

## Pipeline Stages
- Stage 1: Sourced
- Stage 2: Connected
- Stage 3: Replied
- Stage 4: Qualified
- Stage 5: Call Booked
- Stage 6: Proposal Sent
- Stage 7: Won / Lost

## Week 1 Cadence (Agency-heavy)

### Day 1-2
- Source 40 agency contacts/day.
- Send 25 connection requests/day.
- Send 15 first messages/day to accepted connections.

### Day 3-4
- Source 40 agency contacts/day.
- Send 20 connection requests/day.
- Send follow-up 1 to non-responders from Day 1.
- Continue first messages to new accepts.

### Day 5-7
- Source 25 agency contacts/day.
- Send follow-up 2 for warm non-responders.
- Move qualified leads to call scheduling.

## Week 2 Cadence (Brand expansion)

### Day 8-10
- Source 30 brand contacts/day and 15 agency contacts/day.
- Send 20 connection requests/day.
- Send first messages to accepted connections.
- Prioritize K-beauty and DTC teams with active launch cycles.

### Day 11-14
- Continue brand-heavy sourcing.
- Run follow-up 1 and follow-up 2 windows.
- Book calls and prep proposal notes using sample brief references.

## Daily Throughput Targets
- New sourced leads: 30-50
- Connection requests: 20-25
- First outbound messages: 12-20
- Follow-ups: 10-20

## Metrics Dashboard
- Acceptance rate = accepted connections / requests sent
- Reply rate = replies / first messages sent
- Qualified call rate = qualified leads / replies
- Proposal rate = proposals sent / qualified calls

## Target Ranges
- Acceptance rate: 25-40%
- Reply rate: 15-30%
- Qualified call rate: 30-50%
- Proposal rate: 40-70%

## Failure Handling

### Low acceptance
- Tighten title targeting.
- Reduce generic connection notes.
- Prioritize active posters and commenters.

### Low reply
- Shorten opener to one pain point and one proof artifact.
- Move pricing lower in message if response drops too early.
- Improve category relevance in first line.

### Low meeting conversion
- Send sample brief before call request.
- Add one-line qualification question tied to active campaign volume.
- Offer pilot framing first, retainer second.

## Weekly Review
- End of Week 1: evaluate message variant performance.
- End of Week 2: promote winning sequence into default playbook.
